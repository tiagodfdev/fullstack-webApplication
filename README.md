# fullstack-webApplication

## Front-end Stack
- HTML
- CSS
- Javascript
- ReactJS
- ChakraUI
- Axios

## Back-end Stack:
- TypeORM
- MySQL
- Node.js
- Docker

## How to use

- Create back-end/.env and put your MD5 hash
- Run on /back-end "yarn"
- Run on /back-end "docker-compose up --build"
- Run on /front-end "yarn"
- Run on /front-end "yarn start"
